Задача классификации – одна из самых распространенных задач машинного обучения. Она относится к типу задач обучения с учителем.

Классификация в машинном обучении – это задача определения принадлежности объектов к одному из заранее определенных классов на основании его признаков.


**Метод опорных векторов или SVM (Support Vector Machines)**


Для того, чтобы приступить к решению, нужно провести предобработку (заменить пропущенные значения, закодировать категориальные признаки) и разделить данные на обучающую и тестовую выборки.
По полученным данным нужно определить матрицу признаков X и вектор ответа y. Далее, с помощью следующей строки кода, мы можем разделить наши данные на наборы для обучения и тестирования:
- X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.4, random_state=1)
- Переходим к решению задачи классификации. Метод опорных векторов реализован в библиотеке Scikit-Learn.
- from sklearn.svm import SVC 
- Создадим объект метода:
- svc = SVC()
- Для того, чтобы обучить модель используем метод fit().
- svc.fit(X_train, y_train)
- Для предсказания вызываем метод predict().
- y_svc_pred = svc.predict(X_test)


**Метод k-ближайших соседей**
Пример реализации метода в библиотеке Scikit-Learn в Python:
- from sklearn.neighbors import KNeighborsClassifier
- k_nbrs = KNeighborsClassifier()
- В методе можно задавать параметры, например, n_neighbors — количество ближайших соседей, используемых для предсказания. По умолчанию он равен 5.
- Аналогично предыдущему методу сначала обучаем модель и затем уже предсказываем значения.
- k_nbrs.fit(X_train, y_train)
- y_knbrs_pred = k_nbrs.predict(X_test)


**Дерево решений**
Пример реализации метода в библиотеке Scikit-Learn в Python:
- from sklearn.tree import DecisionTreeClassifier
- clf_tree = DecisionTreeClassifier()
- Для улучшения качества предсказаний в этом методе так же можно задавать разные параметры, например, max_depth – максимальная глубина дерева.
- Далее необходимо обучить модель и затем получить предсказания:
- clf_tree.fit(X_train, y_train)
- y_tree_pred = clf_tree.predict(X_test)


**Бинарная логистическая регрессия**
Это метод построения линейного классификатора, позволяющий оценивать вероятности принадлежности объектов классам.
- Реализация метода в библиотеке Scikit-Learn в Python:
- from sklearn.linear_model import LogisticRegression
- Создадим объект логистической регрессии:
- lreg = LogisticRegression()
- Далее нужно обучить модель на тренировочной выборке и сделать прогнозы на тестовом наборе.
- lreg.fit(X_train, y_train)
- y_lreg_pred = lreg.predict(X_test)


**Оценка качества классификации**
- Accuracy (доля правильных ответов) показывает долю правильно классифицированных объектов в выборке.
- from sklearn.metrics import accuracy_score
- accuracy_score(y_test, predictions)
- Recall (полнота) показывает долю найденных объектов класса 1 к общему числу объектов класса.
- from sklearn.metrics import recall_score
- recall_score(y_test, predictions)
- Precision (точность) показывает долю объектов класса среди объектов, выделенных классификатором.
- from sklearn.metrics import precision_score
- precision_score(y_test, predictions)
- Для общей оценки качества классификатора часто используют F₁ меру — среднее гармоническое между precision и recall:
- from sklearn.metrics import f1_score
- f1_score(y_test, predictions)


**Задача кластеризации** относится к типу задач обучения без учителя. Это задача разбиения выборки объектов на непересекающиеся подмножества так, чтобы объекты внутри кластера были похожи, а объекты разных кластеров существенно отличались.


**Метод k-means реализован в библиотеке Scikit-Learn:**
- from sklearn.cluster import KMeans
- Создадим объект метода:
- kmeans = KMeans(n_clusters=2)
- В аргументах нужно указать n_clusters – количество кластеров, которые нужно сформировать. По умолчанию это значение равно 8.
- Далее с помощью fit() и predict() обучаем модель на наших данных и предсказываем распределение их на указанное количество кластеров.
- kmeans.fit(df)
- y_kmeans = kmeans.predict(df)


Для вычисления метрик используется модуль metrics библиотеки Scikit-Learn.
- from sklearn import metrics
Гомогенность и полнота вычисляются с помощью методов homogeneity_score() и completeness_score() соответственно.
V-мера комбинирует в себе гомогенность и полноту и может быть вычислена с помощью метода v_measure_score().


В машинном обучении при работе с данными имеются два типа признаков: количественные (численные) и категориальные (некоторые свойства, которые не имеют численного значения).
Количественные признаки удобно обрабатывать и использовать в моделях.
Категориальные признаки требуют дополнительной обработки -  кодирования (преобразования в числовое представление). Это нужно для корректной работы моделей прогнозирования. Например, в линейных моделях на вход можно передавать только численные признаки.
Два наиболее популярных Encoder’а – Label Encoder и One-Hot-Encoder.


- from sklearn.preprocessing import LabelEncoder
- Создадим объект метода
- labelencoder = LabelEncoder()
- Далее выбираем к какой колонке применить данный метод. Пусть название колонки будет содержать переменная col.
- data[col] = labelencoder.fit_transform(data[col])
